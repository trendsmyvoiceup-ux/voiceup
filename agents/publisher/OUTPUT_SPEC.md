# Publisher — Output Spec

## Output

The same `output/battles/<slug>/` files (`caption.txt`, `hashtags.txt`,
`image_prompt.txt`, `video_prompt.txt`), confirmed or lightly revised for
publication-readiness. No new files, no new locations.

## What the Publisher must NOT output

- Any actual call to an external platform API (TikTok or otherwise).
- Any generated image or video file.
- Any change to `apps/web` or `comparisons.ts`.

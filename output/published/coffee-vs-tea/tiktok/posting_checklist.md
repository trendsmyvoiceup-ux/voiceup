# TikTok Posting Checklist — Coffee vs Tea

## Before recording / editing
- [ ] Read script.txt — know the hook, setup, and CTA by heart
- [ ] Confirm video_prompt.txt describes the visual you will create
- [ ] Caption fits within 2,200 characters (check caption.txt)
- [ ] Hashtag count is ≤ 30 for optimal reach (check hashtags.txt)

## Before uploading
- [ ] Video is vertical 9:16, 6–15 seconds
- [ ] First frame is visually arresting — no black leader
- [ ] On-screen text matches the script hook
- [ ] Update battle_link.txt with the live domain
- [ ] Add the live link to TikTok bio (or Link-in-Bio tool)

## TikTok upload settings
- [ ] Privacy: Public
- [ ] Allow comments: Yes
- [ ] Allow Duet: Yes
- [ ] Allow Stitch: Yes
- [ ] Schedule or post immediately

## After posting
- [ ] Copy the TikTok post URL
- [ ] Mark status as 'posted' in the admin TikTok queue
- [ ] Monitor first-hour: views, likes, shares, comments
- [ ] Reply to comments within 24 h
- [ ] Note top-performing hook for future battles
